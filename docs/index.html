Vorgehensweise

Wir haben uns in 2 Gruppen aufgeteilt. Dilara, Benedict und Johanna haben sich mit dem Prototyp und Javascript beschäftigt. Aaron und Marcel haben in der Zeit die Konzeptseite erstellt.

Prototyp

Zuerst haben wir mittels Google API eine Google Map eingebunden. (https://developers.google.com/maps/documentation/javascript/examples/map-simple) Danach wurde die Standortermittlung hinzugefügt. (https://developers.google.com/maps/documentation/javascript/examples/map-geolocation) Als nächstes wollten wir die Hotelsuche einfügen. Unsere Vorlage: https://developers.google.com/maps/documentation/javascript/examples/places-autocomplete-hotelsearch Dieses Beispiel wollten wir so ändern, dass man den Standort nicht extra eintippen muss, sondern der aktuelle Standort genutzt wird. Hierbei gab es Schwierigkeiten. Wir wussten nicht, wie wir die aktuelle Position einbinden sollte, sodass er automatisch sucht, da das Programm auf eine Eingabe wartet. Wahrscheinlich muss erst einmal der Wert umgewandelt werden (von den Koordinaten zu einem Stadtnamen). Dann muss dieser Wert als Standardwert benutzt werden. Es würde dann ja keine Input Felder mehr geben. Später hat Benedict alleine versucht das Problem zu lösen, während Dilara die Funktion im Prototyp eingebunden hat. Johanna hat eine Webcam in der 4. Section eingebaut und die Funktion auf der Konzeptseite reingestellt.

Konzeptseite

Wir haben uns an Wunderlist.com orientiert, also oben eine Navigation und ein Bild als Eyecatcher. Beim runterscrollen, gibt es verschiedene Sektionen, die die volle Breite des Bildschirms nutzt. Schwierigkeiten gab es mit Live View bei Brackets. Bei der Live View wird die Seite richtig angezeigt, manuell über die Index Datei nicht mehr. Aaron hat das Layout der Website erstellt, während Marcel sich um die Navigation gekümmert hat.

Konzepseite für die Challenge 2017. Eine Studienarbeit im Modul "Web-basierte Anweundungen 1: WWW-Technologien" im Studiengang Medieninformatik Bachelor der TH Köln

Studenten: Aaron Joel Kaemmerer-Karsten, Benedict Weinbach, Dilara Güven, Johanna Mayer und Marcel Friedrich
